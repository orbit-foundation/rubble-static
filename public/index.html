Ready to start static developing? Just get started by editing the public folder.

Deploying your files to a localhost server

We've got two ways ready for you. First, please run npm install to get everything set up.

Option 1: ParcelJS

This is the easier option, but might be less functional. Just go to the public folder, and run parcel index.html in a terminal window. You can then browse to localhost:1234 and, the magic comes alive!

Option 2: Express

For a bit of extra customisation, we've got a deployment JS file in the root folder. Just run node deploy.js in the root folder and browse to localhost:5000 (change this in deploy.js).

And you're off!
